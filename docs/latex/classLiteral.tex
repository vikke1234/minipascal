\doxysubsection{Detailed Description}
A literal, a digit or string, since booleans are not part of the spec, they can not exist as a literal. Internally the symbol table contains Literals, so they\textquotesingle{}re possible to store in a \mbox{\hyperlink{classLiteral}{Literal}} but can\textquotesingle{}t be created without an expression in the language.
